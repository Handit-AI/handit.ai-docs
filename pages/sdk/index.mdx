# SDK Documentation

Handit.ai provides official SDKs for popular programming languages to make it easy to integrate with our platform.

## Available SDKs

### Python SDK

```bash
pip install handit-sdk
```

[View Python SDK Documentation](/sdk/python)

### JavaScript SDK

```bash

```

```bash

```

## Common Features

All SDKs provide:

- Authentication handling
- Model management
- Deployment management
- Prediction making
- Error handling
- Type definitions
- Async support

## Examples

### Python Example

```python
from handit import Client

client = Client(api_key="your-api-key")

# Create a model
model = client.models.create(
    name="my-model",
    framework="pytorch",
    version="1.0.0"
)

# Make a prediction
prediction = client.predictions.create(
    model_id=model.id,
    input_data={"text": "Hello, world!"}
)
```

### JavaScript Example

```javascript
import { Client } from '@handit/sdk';

const client = new Client({ apiKey: 'your-api-key' });

// Create a model
const model = await client.models.create({
    name: 'my-model',
    framework: 'tensorflow',
    version: '1.0.0'
});

// Make a prediction
const prediction = await client.predictions.create({
    modelId: model.id,
    inputData: { text: 'Hello, world!' }
});
```

## Best Practices

- Use environment variables for API keys
- Implement proper error handling
- Use async/await for better performance
- Follow the SDK's type definitions
- Keep SDK versions up to date
