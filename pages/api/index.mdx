# API Reference

Welcome to the Handit.ai API documentation. Our API is RESTful and uses standard HTTP response codes, authentication, and verbs.

## Authentication

All API requests require authentication using an API key. Include your API key in the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Base URL

```
https://api.handit.ai/v1
```

## Endpoints

### Models

#### List Models
```http
GET /models
```

#### Get Model
```http
GET /models/{model_id}
```

#### Create Model
```http
POST /models
```

#### Update Model
```http
PUT /models/{model_id}
```

#### Delete Model
```http
DELETE /models/{model_id}
```

### Deployments

#### List Deployments
```http
GET /deployments
```

#### Create Deployment
```http
POST /deployments
```

#### Get Deployment Status
```http
GET /deployments/{deployment_id}
```

### Predictions

#### Make Prediction
```http
POST /predictions
```

## Rate Limits

- 100 requests per minute for free tier
- 1000 requests per minute for pro tier
- 10000 requests per minute for enterprise tier

## Error Handling

The API uses conventional HTTP response codes:

- 2xx: Success
- 4xx: Client Error
- 5xx: Server Error
