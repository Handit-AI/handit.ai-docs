# Quickstart Guide

Get started with Handit.ai in minutes. This guide will walk you through the basic setup and your first project.

## Prerequisites

- Node.js 16.x or later
- npm or yarn
- A Handit.ai account

## Installation

1. Install the Handit.ai CLI:

```bash
npm install -g @handit/cli
```

2. Login to your account:

```bash
handit login
```

## Create Your First Project

1. Initialize a new project:

```bash
handit init my-first-project
cd my-first-project
```

2. Configure your project:

```bash
handit config
```

3. Deploy your first model:

```bash
handit deploy
```
