# Frequently Asked Questions

## General Questions

### What is Handit.ai?
Handit.ai is an AI platform that helps teams build, train, and deploy machine learning models at scale. We provide tools for model management, deployment, and monitoring.

### How do I get started?
Check out our [Quickstart Guide](/guides/quickstart) to begin using Handit.ai. You'll need to create an account and install our CLI tool.

### Is there a free tier?
Yes! We offer a free tier with basic features. Check our [Pricing Page](/pricing) for details.

## Technical Questions

### What programming languages are supported?
We support Python, JavaScript, Go, and Ruby through our official SDKs. See our [SDK Documentation](/sdk) for details.

### How do I handle API authentication?
All API requests require an API key. Include it in the `Authorization` header:
```bash
Authorization: Bearer YOUR_API_KEY
```

### What are the rate limits?
- Free tier: 100 requests/minute
- Pro tier: 1000 requests/minute
- Enterprise tier: 10000 requests/minute

## Billing Questions

### How is pricing calculated?
Pricing is based on:
- Number of models
- API calls
- Compute resources
- Storage usage

See our [Pricing Page](/pricing) for detailed information.

### Can I upgrade/downgrade my plan?
Yes, you can change your plan at any time. Changes take effect at the start of your next billing cycle.

### Do you offer refunds?
We offer refunds within 14 days of purchase if you're not satisfied with our service.

## Security Questions

### Is my data secure?
Yes, we take security seriously. We use:
- End-to-end encryption
- Regular security audits
- SOC 2 compliance
- GDPR compliance

### Where is my data stored?
Data is stored in AWS data centers in the region you select. We support multiple regions for compliance and performance.

### Do you offer custom security features?
Yes, enterprise customers can request custom security features. [Contact Sales](/contact) for details.
